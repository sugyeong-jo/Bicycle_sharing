# The Gravity Model 
- 이 논문은 중력 모델을 이용해서 location recommender system을 제안 함
- 이 모델은 공간적 간섭 (예를들면, 교통, 이민, 무역 물동량)에 사용되고 뉴튼역학을 기반으로 함
- Mass와 Distance와 같은 잠재 변수를 포함한다.
